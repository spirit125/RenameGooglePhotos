## RenameGooglePhotos專案

## 功能說明  

將所有 **Google 相簿** 內的照片皆下載後，發現檔名是沒有任何意義的名稱，難以閱讀。  
檢查裡面檔案內容後發現每個圖片皆有對應的 **.json** 檔記錄一些圖片資訊。  
所以決定寫這支程式，對指定資料夾內的所有圖檔尋找對應的 **.json** 檔，然後從中取得 **拍攝日期**，  
若找不到 **.json** 檔就讀檔案本身記錄的 **拍攝日期**，  
將圖片檔名改為 **yyyyMMdd_HHmmss[原檔名]** 的格式。  

## 開發環境建立  

專案類型為 **WPF應用程式** ，使用 **.NET 6.0** 版本。  
執行 **git clone** 後，  
使用Visual Studio 2022 **開啟專案或解決方案** 讀取 **.sln檔**  

### 安裝 NuGet 套件

FolderBrowserEx 1.0.1：<https://github.com/evaristocuesta/FolderBrowserEx>  

## 使用方法  

執行 EXE 視窗程式。  
點擊 **選擇資料夾** 輸入 **Google 相簿** 下載後的圖檔路徑。  
再點擊 **開始進行檔案更名** 就開始執行。  
執行完如果想復原檔名，可再點擊 **恢復檔案名稱**。
